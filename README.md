# H
H是一款强大的资产收集管理平台(目前还在开发中)

预计实现以下功能

- [x] 域名扫描
- [x] 端口扫描
- [x] web站点扫描
- [x] 目录扫描
- [ ] 漏洞扫描(nuclei, 自定义poc, xray+rad) 
- [ ] host碰撞插件
- [ ] apk域名提取插件
- [ ] 备案信息查询
- [ ] fofa查询
- [ ] 多账号管理
