# H
H是一款强大的资产收集管理平台

实现以下功能

- [x] 域名扫描
- [x] 端口扫描
- [x] web站点扫描
- [x] 目录扫描
- [x] 漏洞扫描(nuclei, 自定义poc, xray+rad) 
- [x] host碰撞插件
- [x] apk域名提取插件
- [x] 备案信息查询
- [ ] fofa查询
- [ ] 多账号管理
